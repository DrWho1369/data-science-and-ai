ğŸ”¹ Writing Unit Tests in Python (`unittest`)

Python has a built-in module called `unittest` that allows us to test our code.

# Function to test

    def test_addition(self):
        # Check if add(2, 3) correctly returns 5

# Run tests when the file is executed

ğŸ”¹ Test-Driven Development (TDD)

TDD is a software development approach where you write tests before writing code.

Steps in TDD:

  1. Write a failing test (because the function doesn't exist yet).
  2. Write just enough code to make the test pass.
  3. Refactor (improve) the code to make it cleaner.
