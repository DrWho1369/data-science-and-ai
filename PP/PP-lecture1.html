🔹 What is a Program?

A program is a set of instructions that a computer executes to perform a specific task. Programs consist of basic operations like input, output, arithmetic, conditional execution, and loops.

🔹 Variables & Data Types

  • int: Whole numbers (e.g., 42)
  • float: Decimal numbers (e.g., 3.14)
  • str: Text data (e.g., "Hello World!")
  • bool: True/False values

🔹 Boolean Expressions & Logical Operators

  • == (equals), != (not equal), >, <, >=, <=
  • and: True only if both conditions are True
  • or: True if at least one condition is True
  • not: Negates a Boolean expression

💻 Key Code / Functions

🔹 Assigning Variables

# Assigning values to variables
age = 25
name = "Alice"
pi = 3.1415

🔹 Boolean Expressions

# Checking conditions
x = 5
print(x > 3)  # True
print(x == 5) # True
print(x != 10) # True

🔹 Logical Operators

# Using and, or, not
is_student = True
has_pass = False

print(is_student and has_pass)  # False
print(is_student or has_pass)   # True
print(not is_student)           # False

🔹 If Statements

# Basic if statement
if x > 0:
    print("x is positive")

🔹 If-Else Statement

# If-else example
if x % 2 == 0:
    print("Even number")
else:
    print("Odd number")
