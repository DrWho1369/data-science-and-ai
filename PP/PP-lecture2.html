ğŸ”¹ What is Iteration?

Iteration means repeating a set of instructions until a condition is met.

ğŸ”¹ While Loop

A while loop runs as long as the condition is True:

n = 5
while n > 0:
    print(n)
    n -= 1
print("Blastoff!")

ğŸ”¹ For Loop

A for loop iterates over a sequence:

for i in range(1, 6):  
    print(i)

ğŸ”¹ Break Statement

The break statement stops a loop early:

while True:
    name = input("Enter name (type 'exit' to stop): ")
    if name == 'exit':
        break
    print("Hello, " + name + "!")

ğŸ”¹ What Are Functions?

A function is a named block of code that performs a task.

def greet():
    print("Hello!")

ğŸ”¹ Functions with Parameters

Functions can take input values:

def greet(name):
    print("Hello, " + name + "!")

ğŸ”¹ Functions with Return Values

Functions can return results:

def square(n):
    return n * n

result = square(4)
print(result)  # Output: 16
