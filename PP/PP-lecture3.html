ğŸ”¹ What Are Collections?

Collections are used to store multiple pieces of data in a structured way. Different types of collections have different rules for order, mutability, and uniqueness.

ğŸ”¹ Lists (Ordered & Mutable)

A list is a flexible collection that stores multiple elements in a specific order. You can modify lists after they are created.

my_list = [1, 2, 3, "Python", True]
print(my_list[0])  # Output: 1
my_list.append(4)   # Adds 4 to the end

ğŸ”¹ Dictionaries (Key-Value Pairs)

A dictionary stores data using keys and values. Think of it like a real-life dictionary where each word (key) maps to a definition (value).

my_dict = {"name": "Alice", "age": 25}
print(my_dict["name"])  # Output: Alice
my_dict["age"] = 26  # Updating a value

ğŸ”¹ Tuples (Immutable & Ordered)

A tuple is just like a list, but cannot be modified once created. This makes tuples faster and safer for storing fixed data.

my_tuple = (10, 20, 30)
print(my_tuple[1])  # Output: 20

ğŸ”¹ Sets (Unordered & Unique)

A set is a collection of unique items, meaning duplicates are automatically removed.

my_set = {1, 2, 3, 4, 4, 5}
print(my_set)  # Output: {1, 2, 3, 4, 5}

ğŸ”¹ Strings (Text Sequences)

A string is an immutable sequence of characters, used for storing text.

my_string = "Hello, World!"
print(my_string[0])  # Output: H

ğŸ”¹ Working with Lists

Lists store multiple items in an ordered sequence.

my_list = [1, 2, 3, 4]
my_list.append(5)  # Adds 5 at the end
my_list.remove(3)  # Removes value 3
print(my_list)  # Output: [1, 2, 4, 5]

ğŸ”¹ Using Dictionaries

Dictionaries allow quick access to values using unique keys.

my_dict = {"apple": 2, "banana": 5}
print(my_dict["banana"])  # Output: 5

ğŸ”¹ Using Sets

Sets store unique values and remove duplicates automatically.

my_set = {1, 2, 3, 3, 4}
print(my_set)  # Output: {1, 2, 3, 4}

ğŸ”¹ String Manipulation

Strings allow easy text processing and manipulation.

my_string = "Python"
print(my_string.upper())  # Output: PYTHON
print(my_string.lower())  # Output: python
