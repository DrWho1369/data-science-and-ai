🔹 What Are Functions?

A function is a reusable block of code that performs a specific task. It prevents code duplication and makes programs easier to manage.

def greet():
    print("Hello, world!")

When you call `greet()`, the function executes its code.

🔹 Parameters & Return Values

Functions can take parameters (inputs) and return values.

def add(a, b):
    return a + b

result = add(3, 5)  # Output: 8

✅ The function takes two values, adds them, and returns the result.

🔹 Memory in Programming (Stack vs. Heap)

Computers use two types of memory storage:

  • 🟢 Stack → Stores function calls, local variables, and primitive data types.
  • 🔵 Heap → Stores complex objects (lists, dictionaries) that persist beyond function calls.

🔹 References & Aliasing

When a variable stores a reference, it points to an object in memory. This means multiple variables can point to the same object.

a = [1, 2, 3]  # Stored in heap memory
b = a  # 'b' now refers to the same list as 'a'
b.append(4)
print(a)  # Output: [1, 2, 3, 4]

✅ Since `b` and `a` both reference the same object, modifying `b` also changes `a`.

🔹 Matrices

A matrix is a two-dimensional structure used to store data in rows and columns. In Python, a matrix is often represented as a list of lists.

matrix = [
    [1, 2, 3],
    [4, 5, 6],
    [7, 8, 9]
]

def function():
    x = 5  # Stored in Stack
    y = [10, 20, 30]  # Stored in Heap

🔹 Understanding References

a = [1, 2, 3]
b = a
b.append(4)
print(a)  # Output: [1, 2, 3, 4]

🔹 Matrices in Python

matrix = [
    [1, 2, 3],
    [4, 5, 6],
    [7, 8, 9]
]

print(matrix[1][2])  # Output: 6
