Revision Notes

1. Introduction to Basic Statistics

Statistics is the study of data collection, analysis, interpretation, and presentation.

2. Key Statistical Measures

  • Mean: The average value.
  • Median: The middle value when sorted.
  • Mode: The most frequently occurring value.
  • Variance: Measures data spread.
  • Standard Deviation: Square root of variance.

3. Probability and Bayes' Theorem

Bayes' Theorem: Used to update probabilities based on new evidence.

Formula:

P(H|E) = (P(E|H) * P(H)) / P(E)

4. Statistical Distributions

  • Normal Distribution: Bell-shaped curve where data clusters around the mean.
  • Binomial Distribution: Models binary outcomes (success/failure).
  • Poisson Distribution: Counts events over a fixed period/space.

WEEK 2 LABS - R Exercises

1. Basic Descriptive Statistics

Define a numeric vector x:


x <- c(10, 2, 15, 6, 4, 9, 12, 11, 3, 0, 12, 10, 9, 7, 11, 10, 8, 5, 10, 6)

Calculate fundamental statistics:

  • Number of Observations:
    • length(x)
  • Sum of All Observations:
    • sum(x)
  • Mean (Average):
    • mean(x)
  • Median (Middle Value):
    • median(x)
  • Mode (Most Frequent Value):
    • table(x)[which.max(table(x))]
  • Five Number Summary:
    • summary(x)

2. Generating Random Vectors & Computing Correlations

Generate a normally distributed vector y with the same mean as x:

y <- rnorm(20, mean(x), 1)

Compute the covariance and correlation between x and y:

cov(x, y)
cor(x, y)

Set a seed for reproducibility and generate another random variable z:

set.seed(100)
z <- rnorm(20, 8, 1)
plot(x, z, main = "Scatter plot of x and z", xlab = "x", ylab = "z")

3. Boxplot Visualization

Create a Boxplot for x and y to visualize distributions:

data <- data.frame(x, y)
boxplot(data, col = c("green", "orange"), main = "Boxplot for x and y")
