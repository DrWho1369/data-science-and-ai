1Ô∏è‚É£ What is a System of Linear Equations?

An SLE consists of multiple linear equations involving the same set of unknown variables.

üîπ Example of an SLE


        2x + 3y = 8
        5x -  y = 4

The goal is to find values of x and y that satisfy both equations.

2Ô∏è‚É£ Types of Solutions for SLEs

  • Unique Solution: When the system has exactly one solution.
  • Infinitely Many Solutions: When the system represents overlapping lines or planes.
  • No Solution: When the equations represent parallel lines that never intersect.

3Ô∏è‚É£ Solving SLEs Using Gauss-Jordan Elimination

The Gauss-Jordan elimination method is a systematic way to solve SLEs by transforming the augmented matrix into row echelon form (RREF).

üîπ Steps:

  1. Convert the system into an augmented matrix.
  2. Use row operations to create a leading 1 in the first column.
  3. Make all other entries in that column zero.
  4. Repeat the process for subsequent columns.
  5. Continue until the matrix is in reduced row-echelon form (RREF).
  6. Extract the solutions from the final matrix.

4Ô∏è‚É£ Eigenvalues and Eigenvectors

Eigenvalues and eigenvectors provide fundamental insights into linear transformations.

üîπ Definition

For a matrix A, an eigenvector v satisfies:


        A * v = Œª * v

where Œª (lambda) is the eigenvalue.

üîπ Finding Eigenvalues

We solve the characteristic equation:


        det(A - ŒªI) = 0

üîπ Finding Eigenvectors

Once eigenvalues are found, we solve:


        (A - ŒªI) v = 0

to get the eigenvectors.

Key Code / Functions

Solving an SLE in Python using NumPy


import numpy as np

A = np.array([[2, -1], [1, 1]])
B = np.array([1, 5])

X = np.linalg.solve(A, B)
print("Solution:", X)

Computing Eigenvalues and Eigenvectors


import numpy as np

A = np.array([[4, -2], [1, 1]])

# Compute eigenvalues and eigenvectors
eigenvalues, eigenvectors = np.linalg.eig(A)

print("Eigenvalues:", eigenvalues)
print("Eigenvectors:\n", eigenvectors)

This computes the eigenvalues and eigenvectors of a given matrix using NumPy.
