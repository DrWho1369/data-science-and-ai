Revision Notes

Vectors

A vector is an ordered list of numbers that can be interpreted as a direction and magnitude in space. For example:


        v = [3, 4]  # A vector in 2D space

The magnitude (or length) of a vector is given by:


        |v| = sqrt(3² + 4²) = 5

Vectors can be added, subtracted, or scaled by multiplying them by a scalar.

Dot Product and Orthogonality

The dot product of two vectors a and b is calculated as:


        a · b = a₁b₁ + a₂b₂ + ... + aₙbₙ

If the dot product is zero, the vectors are orthogonal (perpendicular).

Matrices

A matrix is a rectangular array of numbers that can be used for linear transformations and solving systems of equations. Example:


        A = | 1  2 |
            | 3  4 |

The identity matrix acts as the neutral element in multiplication:


        I = | 1  0 |
            | 0  1 |

Determinants and Inverses

The determinant of a 2×2 matrix A is:


        det(A) = (1×4) - (2×3) = -2

If det(A) ≠ 0, the matrix has an inverse, meaning it can be used to solve linear systems.
