üìå What is Probability?

Probability is a branch of mathematics that quantifies uncertainty. It allows us to analyze random events and estimate the likelihood of different outcomes occurring.

üîπ Example:

  • A coin flip has two possible outcomes: Heads or Tails.
  • Probability helps us understand how often each outcome should occur over many trials.

Probability bridges the gap between what we know and what is unknown about the future.

üìå Sample Space & Events

üü¢ 1. Sample Space (S)

The sample space is the set of all possible outcomes of a random experiment.

Experiment Sample Space (S)
Flip a coin S = {Heads, Tails}
Roll a die S = {1,2,3,4,5,6}
Draw a card S = (all 52 cards in a deck)
Customer payment method S = {Credit, Debit, Cash, Check}

üü¢ 2. Events & Simple vs. Compound Events

  • An event is any subset of outcomes in the sample space.
  • Simple Event: An event that consists of only one outcome (e.g., Rolling a 4).
  • Compound Event: An event that consists of multiple outcomes (e.g., Rolling an even number: {2, 4, 6}).

üìå Three Types of Probability

Probability can be assigned in three distinct ways:

1Ô∏è‚É£ Empirical Probability (Experimental Approach)

  • Based on observed data from experiments.
  • Formula:
P(A) = (Number of times event A occurred) / (Total number of trials)

Example: A company surveys 500 customers. If 200 prefer Brand X, then:

P(Brand X) = 200 / 500 = 0.4

2Ô∏è‚É£ Classical Probability (Theoretical Approach)

  • Used when all outcomes are equally likely.
  • Formula:
P(A) = (Number of favorable outcomes) / (Total number of outcomes in S)

Example: Rolling a die ‚Üí Probability of rolling a 3:

P(3) = 1 / 6

3Ô∏è‚É£ Subjective Probability

  • Based on personal judgment or expert opinion.

Example: A doctor estimates a 70% chance of recovery based on experience.

üìå Probability Rules

1Ô∏è‚É£ Rule 1: Probability Values

  • Probability of any event P(A) must be between 0 and 1:
0 ‚â§ P(A) ‚â§ 1
  • P(A) = 0 ‚Üí Impossible event (e.g., Rolling a 7 on a six-sided die).
  • P(A) = 1 ‚Üí Certain event (e.g., Rolling a number between 1 and 6).

2Ô∏è‚É£ Rule 2: The Sum of Probabilities

P(A) + P(A·∂ú) = 1

üìå Rule 3: Addition Rule for ‚ÄúOR‚Äù Events

General Addition Rule:

P(A ‚à™ B) = P(A) + P(B) - P(A ‚à© B)

Used when A and B are not mutually exclusive.

Special Addition Rule (Mutually Exclusive Events):

P(A ‚à™ B) = P(A) + P(B)

Used when A and B cannot occur together.

üîπ Example:

  • Event A: Rolling an even number (2,4,6) ‚Üí P(A) = 3/6
  • Event B: Rolling a number >4 (5,6) ‚Üí P(B) = 2/6
  • Event A ‚à© B: Rolling a 6 ‚Üí P(A ‚à© B) = 1/6
  • Total Probability:
P(A ‚à™ B) = 3/6 + 2/6 - 1/6 = 4/6

üìå Rule 4: Multiplication Rule for ‚ÄúAND‚Äù Events

General Multiplication Rule:

P(A ‚à© B) = P(A) √ó P(B | A)

Used when A and B are dependent.

Special Multiplication Rule (Independent Events):

P(A ‚à© B) = P(A) √ó P(B)

Used when A and B do not affect each other.

üîπ Example:

Probability of rolling a 6 twice in a row:

P(6 ‚à© 6) = (1/6) √ó (1/6) = 1/36

üìå Conditional Probability

Conditional probability tells us the probability of event A occurring given that B has already occurred.

Formula:

P(A | B) = P(A ‚à© B) / P(B)

üîπ Example:

If 5% of students at a university are in the math club, and 2% are in both the math and chess club, then the probability that a random math club student is also in the chess club is:

P(Chess | Math) = P(Math ‚à© Chess) / P(Math) = 0.02 / 0.05 = 0.4

üìå Bayes' Theorem

Bayes' Theorem is used to update probabilities when given new information.

Formula:

P(B | A) = (P(A | B) √ó P(B)) / P(A)

  • If sick: Tests positive 90% of the time.
  • If healthy: Tests positive 5% of the time.

If 1% of people are actually sick, then:

P(Sick | Positive) = (0.9 √ó 0.01) / ((0.9 √ó 0.01) + (0.05 √ó 0.99)) = 0.16

Even if a test is positive, there‚Äôs only a 16% chance the person is actually sick!
